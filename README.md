# Lobster Buffet

Lobster Buffet is a public open-source software project for shared OpenClaw process-provider operations.

It is the operational/provider-side companion to Lobster Portage. Lobster Portage packages and distributes reusable OpenClaw customization assets; Lobster Buffet exposes portable process operations as deterministic provider/tool calls that multiple OpenClaw instances can use through local adapters.

## Naming Note

`lobster-buffet` is memorable and fits the Lobster Portage family, but it is whimsical for public infrastructure. Treat it as the working name until a naming pass confirms whether the public package should keep the name or adopt a plainer one.
